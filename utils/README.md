# Utilities

Key modules
- logging_utils: central logging and exception helpers
- io: CSV schema, append and cache checks, hashing, runtime versions
- preprocess: unified preprocessing for fair comparisons
- ov_convert/ov_utils: OpenVINO conversion and device queries
- infer_ort/infer_openvino/infer_torch: inference helpers per backend
- device_utils: device detection and naming
- energy: NVML energy measurement helpers

See [overview](../docs/overview.md) and [reproducibility](../docs/reproducibility.md) for how these pieces fit together.
